Get StartedPlayground

Type Safety

Catch type errors at compile time, not runtime. Add types gradually to your existing Ruby code.

Zero Runtime

Types are erased during compilation. The output is pure Ruby that runs anywhere.

RBS Generation

Automatically generates .rbs signature files for integration with Steep, Ruby LSP, and more.

TypeScript-inspired

Familiar syntax for TypeScript developers. Union types, generics, interfaces, and more.

Great DX

VS Code and Neovim support with syntax highlighting, LSP, and real-time error reporting.

Gradual Adoption

Start with one file. Mix typed and untyped code. Migrate at your own pace.

See it in action

Write typed Ruby, compile to standard Ruby with generated type signatures.

# hello.trb
def greet(name: String): String
  "Hello, #{name}!"
end

def add(a: Integer, b: Integer): Integer
  a + b
end

# Union types
def parse_id(id: String | Integer): String
  id.to_s
end

# Generics
def first<T>(arr: Array<T>): T | nil
  arr[0]
end

Quick Start

1

Install T-Ruby

gem install t-ruby
2

Create a .trb file

hello.trb
def greet(name: String): String
  "Hello, #{name}!"
end

puts greet("World")
3

Compile and run

trc hello.trb
ruby build/hello.rb
